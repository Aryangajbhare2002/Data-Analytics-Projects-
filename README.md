# 🚗 Car Sales Dashboard – Power BI Project

## 📌 Project Overview
This project presents an **interactive Power BI dashboard** designed to analyze car sales data across different brands and models. The dashboard provides a clear and dynamic view of sales performance, helping users explore trends and make data-driven decisions.

---

## 🎯 Objectives
- Analyze car sales by **brand and model**
- Understand customer preferences based on **fuel type** and **transmission**
- Provide an interactive experience using filters and visualizations
- Present key insights in a simple and visually appealing format

---

## 🛠️ Tools & Technologies Used
- **Power BI Desktop**
- **Power Query** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – Calculated measures and KPIs  
- **Data Source** – *(Excel / CSV — update if needed)*  

---


---

## 📊 Key Features
- 🔍 **Interactive Slicers**  
  - Filter data by **Brand**, **Model**, **Fuel Type**, and **Transmission Type**

- 📌 **KPI Cards**  
  - Display total sales, number of cars sold, and other key metrics

- 📈 **Charts & Visualizations**
  - Bar/Column charts for comparing brand and model performance  
  - Pie & Donut charts for distribution analysis (fuel type, transmission)

- ⚡ **Dynamic Dashboard**
  - Real-time filtering and cross-visual interaction  

---

## 📈 Dashboard Insights
- Identification of **top-selling car brands and models**  
- Comparison of **fuel type preferences** (Petrol, Diesel, Electric, etc.)  
- Analysis of **transmission trends** (Manual vs Automatic)  
- Overall sales distribution and performance patterns
